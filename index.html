TweetPortfolio: Help

Welcome to TweetPortfolio!

Thank you for using our service!
Input your query into the search bar on the left of the screen, press search, and see a list of tweets just as you would in your normal Twitter feed. If you would like to change the order or the types of tweets in the feed, just change the options after your search.

If you see a tweet you like in your feed, press the add button and select the folder you want to add the tweet to. If you want to see more about that tweet, click the Twitter button to open a new window on the official Twitter page. The Twitter button will be green when you return, so you can see where you are. If you want to make a new folder press the New Folder button on the buttom right of your screen. Finally if you want to edit your folders or their contents, press the edit folder button, which will allow you to change folder names and remove tweets.
